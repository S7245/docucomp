①【当天的时间】，直接显示24小时制的具体小时分钟，如：10:36，18:35；
②【前1～2天的时间】，显示为：昨天/前天 + 24时，如：昨天 14:28， 前天10:48；
③【3～5天内的时间】，显示为：三天前/四天前/五天前（不显示具体时间）；
④【超过5天的时间】，则显示具体的日期，不显示时间，如：10-02，08-12；
⑤【非当年的时间】，则在显示年月日的具体日期，如：2018-12-16。
