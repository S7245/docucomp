# 间距

## 网格系统

点网格系统是一种有助于整齐的对齐和组织设计中元素的设计方法。基本规则是这些元素之间的间距都是 4 的倍数，例如 4、8、12、16 等。应用场景包括布局间距、组件与元素间距、组件大小等等。

**一致性**: 所有元素都按照相同的间距设计，可以使界面看起来更加整齐有序，用户能够更容易地预测和理解界面布局，从而提升用户体验。\
**灵活性**: 无论是小型设备还是大型显示器，4 点网格系统都能灵活调整，保持一致的视觉效果，简化了响应式设计的过程。\
**提升效率**: 设计师在创作时，有一个明确的间距和尺寸参考，减少了设计时间。同时，开发者可以更容易地将设计转换为代码，减少了沟通成本和误差。
