ProLayout - 高级布局​

ProLayout 可以提供一个标准又不失灵活的中后台标准布局，同时提供一键切换布局形态、自动生成菜单等功能。 与 PageContainer 配合使用可以自动生成面包屑、页面标题，并且提供低成本方案接入页脚工具栏。

Mac 安装全局 UMI​

# 全局安装UMI
yarn global add umi

#获取全局 global bin 路径
yarn global bin

# 打开 .bash_profile 文件
open ~/.bash_profile

在 .bash_profile 文件中添加下面一行。
export PATH="$PATH:yarn global bin"

source ~/.bash_profile
