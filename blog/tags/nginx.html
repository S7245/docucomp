October 30, 2023 · 3 min read
Sam
--

安装​

安装环境：

yum -y install gcc 
yum -y install zlib zlib-devel 
yum -y install pcre pcre-devel
yum -y install openssl openssl-devel

nginx 找到Mainline version查看版本，上传到服务器：/usr/local/，

cd /usr/local/
wget https://nginx.org/download/nginx-1.25.3.tar.gz

# 解压
tar -zxvf nginx-1.25.3.tar.gz

# 把 nginx-1.25.3 文件名改为： nginx

# 编译前检查命令
./configure --prefix=/usr/local/nginx
# 编译
make & make install

创建 logs 文件：

cd /usr/local/nginx
mkdir logs

操作：

cd /usr/local/nginx/sbin

# 启动 nginx
./nginx

# 关闭 nginx
./nginx -s stop

# 重启
./nginx -s reload

开机启动 nginx：

vim /etc/rc.local
# 最底部增加这一行
/usr/local/nginx/sbin/nginx

启动后查看进程：

# 查询进程号
ps -ef|grep nginx

# 停止进程
kill -QUIT 进程号

# 快速停止
kill -TERM 主进程号

# 强制停止
kill -9 主进程号

QA：80端口被占用，需要解除

# 查看端口
netstat -ntlp
